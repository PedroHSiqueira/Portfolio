Controle de Chamdos

Esse sistema pode ser útil para gerenciar solicitações de suporte, atendimento ao cliente ou tarefas internas em uma equipe. A estrutura básica do HTML consiste em uma tabela para exibir as informações dos chamados e elementos de formulário para adicionar novos chamados, já o CSS é responsável por estilizar o controle de chamados, definindo propriedades como fonte, margens e alinhamentos

Calculadora de IMC

Permite aos usuários calcular seu IMC com base em sua altura e peso, a interface da calculadora é construída usando HTML e CSS para criar os elementos visuais, como campos de entrada para a altura e o peso, botões para calcular o IMC e exibir o resultado, já o JavaScript é responsável por adicionar a funcionalidade à calculadora
